# Ассеты бренда для КП

Сюда положить файлы заказчика (PNG с прозрачным фоном — лучше всего):

- `logo.png` — логотип (дефолтный, если у сметы нет своего загруженного). Используется в кружке.
- `stamp.png` — печать ИП.
- `signature.png` — подпись.

После добавления файлов реквизиты (текст) правятся в `../company.ts` — там плейсхолдеры
`ИП —`, `ИНН —` и т.д., подставить настоящие из Excel заказчика.

Файлы читаются в data URL на сервере (Puppeteer не резолвит относительные пути),
поэтому именно эти имена. Нет файла — в PDF просто не отобразится, ошибки не будет.
